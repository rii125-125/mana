# Contributing to mana
> [!TIP]
> First of all, thank you for your interest in `mana`! This project is currently in the prototype stage and needs your help.

## We welcome any contributions or issues!
We are particularly seeking contributions such as the following:
- **Bug reports**: Submit via Issues
- **Feature suggestions**: Ideas like "I'd love to see this magical feature"
- **Documentation**: Fixing typos or rewriting explanations for greater clarity
- **Code**: Resolving `TODO` comments or implementing new features

## Getting started with development
Here are the steps to fork this repository and start developing locally:

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/mana.git
    ```
2. Create a branch:
    ```bash
    git checkout -b feature/amazing-feature
    ```
3. Build and test:
    ```bash
    cargo build
    cargo test
    ```
4. Commit and push your changes
